# FontAss

Change Advanced SSA(.ass) subtitle's font

## How to Use

### macOS

1. Download ```ASSA Font Changer.command``` from [Release](https://github.com/AHRI2nd/FontAss/releases).

2. Move ```ASSA Font Changer.command``` to the folder where the font exists.

3. Open 'Terminal' and move to the Script folder

4. Type:
```bash
chmod +x "ASSA Font Changer SUITE-SemiBold.command"
```


Install has been finished.

Double click ```ASSA Font Changer.commnad``` to automatically replace the font

## Font Settings

Basically, this script is set by [Suite-Bold](https://github.com/sun-typeface/SUITE).

Recommend to install [Suite-Variable](https://github.com/sun-typeface/SUITE/releases/) to use.


Otherwise, you can edit ```ASSA Font Changer.command``` by text editor.

On the Line 8, ```target_font = "Font Post Script Name*```
