# FontAss

Change Advanced SSA(.ass) subtitle's font

## How to Use

### macOS

Download ```ASSA Font Changer.command``` from [Release](https://github.com/AHRI2nd/FontAss/releases)

Move ```ASSA Font Changer.command``` to the folder where the font exists

Open 'Terminal' and move to the Script folder

Type
```bash
chmod +x "ASSA Font Changer SUITE-SemiBold.command"
```


Install has been finished.

Double click ```ASSA Font Changer.commnad``` to automatically replace the font
